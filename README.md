# Bank Loan Analytics Dashboard (SQL + Power BI)

📌 Project Overview

This project analyzes bank loan distribution, branch performance, and portfolio health using SQL and Power BI.

The dashboard provides insights into lending patterns, customer segments, and regional performance to support data-driven decision making in financial services.

The goal of this project is to demonstrate data analysis, SQL querying, and interactive dashboard development.


📊 Dashboard Structure

Executive Overview

Total loan applications, funded amount, and received amount,

Loan distribution by product and income segment,

Loan status breakdown (Fully Paid, Current, Charged Off)


Branch & Regional Analysis

Loan amount comparison across regions,

Branch performance analysis,

Product distribution within branches


Customer Insights

Loan distribution by income category,

Loan approval patterns,

Customer demographic insights


🛠 Tools & Technologies

SQL – Data extraction and transformation

Power BI – Interactive dashboard and visualization

Power Query – Data cleaning and transformation

DAX – KPI calculations and analytical measures
